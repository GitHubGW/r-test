# YouTube-Clone-2021

### Routes

#### Global Router

- / -> Home
- /join -> Join
- /login -> Login
- /search -> Search

#### User Router

- /users/:id -> Show User
- /users/logout -> Log Out
- /users/edit -> Edit MY Profile
- /users/delete -> Delete MY Profile

#### Video Router

- /videos/:id -> Show Video
- /videos/:id/edit -> Edit Video
- /videos/:id/delete -> Delete Video
- /videos/upload -> Upload Video
